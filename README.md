# 🎴 바카라 데이터 분석기

빌드 과정 없이 브라우저에서 바로 실행되는 **단일 파일 HTML** 앱입니다.
입력한 과거 게임 기록을 그대로 집계·시각화하고, 베팅 사다리(파레이 복구)
계산기를 제공합니다.

> ⚠️ 이 앱은 입력한 과거 기록을 **집계**할 뿐, 미래를 예측하지 않습니다.
> 바카라의 각 핸드는 독립이라 과거 패턴이 다음 핸드를 바꾸지 않습니다
> (뱅커 ≈45.9%, 플레이어 ≈44.6% 고정). 도박은 장기적으로 손실
> (하우스 엣지 −1.06%)로 수렴합니다. 잃어도 되는 금액만, 손절선을 지키세요.

## 실행 방법

별도 설치가 필요 없습니다. `index.html` 을 브라우저에서 열면 됩니다.
(모바일 화면에 최적화되어 있으며 홈 화면에 추가해 앱처럼 사용할 수 있습니다.)

```bash
# 정적 서버로 실행하려면 (선택)
python3 -m http.server 8000
# 이후 http://localhost:8000 접속
```

## 주요 기능

1. **데이터 입력**: B(뱅커)/P(플레이어)/T(타이) 버튼 또는 텍스트 직접 입력, 마지막 삭제·전체 지우기
2. **빈도·연속 통계**: 총 핸드 수, B/P/T 개수·비율, 최장 연속
3. **패턴 분석**: 직전 1~4핸드 패턴별로 그 다음에 실제로 나온 결과를 집계 (확률 예측이 아님)
4. **베팅 사다리 계산기**: 시작·목표 자금, 기본 베팅, 손절(연패), 커미션을 입력하면 단계별 1차 베팅액과 연패 시 누적 손실을 표로 계산
