# 🎴 바카라 전략 앱

빌드 과정 없이 브라우저에서 바로 실행되는 단일 페이지 웹앱입니다.
바카라 게임 결과를 기록하고, 로드맵을 시각화하며, 여러 베팅·예측 전략에
따라 다음 베팅을 추천하고 자금 변화를 시뮬레이션합니다.

> ⚠️ 이 앱은 **학습 및 기록용 도구**입니다. 바카라는 매 게임이 독립적이며,
> 어떤 전략도 카지노의 하우스 엣지를 이길 수 없습니다. 책임감 있게 이용하세요.

## 실행 방법

별도 설치가 필요 없습니다. `index.html` 을 브라우저에서 열면 됩니다.

```bash
# 정적 서버로 실행하려면 (선택)
python3 -m http.server 8000
# 이후 http://localhost:8000 접속
```

## 주요 기능

- **결과 기록**: 플레이어(P) / 뱅커(B) / 타이(T) 입력, 취소, 전체 삭제
- **로드맵 시각화**
  - 비드 플레이트(Bead Plate): 입력 순서대로 6행 세로 배치
  - 빅 로드(Big Road): 연속 결과 표시 및 6행 초과 시 드래곤 테일 처리, 타이 마크 부착
- **예측 전략**: 항상 뱅커 / 추세 추종 / 역추세 / 다수결
- **베팅 시스템**: 고정(Flat) / 마틴게일 / 파롤리 / 1-3-2-6 / 달랑베르
- **자금 관리**: 시작 자금·기본 베팅 설정, 현재 자금·손익·추천 베팅액 표시
- **통계**: P/B/T 비율, 현재 연속, 추천 적중률

## 데이터 저장

모든 데이터는 브라우저의 `localStorage` 에만 저장되며 서버로 전송되지 않습니다.
