### F. 기타

- **[중요] 언어**: 5개 언어 × 셸 문구 일부만(L509-521). 툴박스 탭·메뉴·주문창은 영어 고정. MT5는 전 UI 다국어. 데모 공개 시 EN 단일로 보고 가면 무방하나 "언어" 셀링포인트로는 미완.
- **[선택] 라이트 테마**: MT5엔 기본 라이트 테마가 없고 색상 커스텀만 있음 — 우리 라이트(L431-441)는 MT5 초과 기능(무해).
- **[중요] 로그인 진입점**: 헤더 아바타 "Sign in"(L484)에 **클릭 핸들러가 없음** — JS 전체에서 .avatar 바인딩 0. 로그인은 다른 페이지 세션에 의존(fxAcct L2036-2081). 실계좌 조회용 공개라면 죽은 Sign in 버튼은 즉시 발각.

## 2) 구현됐지만 MT5와 동작이 다른 것 (디테일 차이)

1. **마우스휠 = 줌** (L1137-1138). MT5 기본은 **휠 = 가로 스크롤**, 줌은 +/−·툴바. 차트 유저 첫 3초에 느끼는 차이. [ESTIMATE: 최신 빌드는 Ctrl+휠 줌 지원]
2. **Auto Scroll / Chart Shift가 전역 토글** (L721-723, L1876-1880). MT5는 **차트별 속성**(활성 차트에만 적용). 나머지 조작(타입/TF/줌/지표)은 차트별로 잘 분리돼 있어(L741-745) 이 둘만 이질적.
3. **계좌 요약 바가 sticky가 아님**: Trade는 tfoot이 `.tbxscroll` 내부(L1663-1670), Exposure도 내부(L1701-1704) — 포지션/노출이 많아 스크롤되면 Balance/Equity/총P/L이 화면 밖으로. thead만 sticky(L320). 실제 MT5도 요약 행이 리스트와 함께 스크롤되지만[ESTIMATE], **사장님 지시("sticky 푸터, 항상 P/L 표시")와 불일치** — `.tbxacct`(L314)를 스크롤 밖으로 빼거나 tfoot에 `position:sticky; bottom:0` 필요.
4. **LIVE 로그인 시 History 탭이 데모 이력 표시**: Trade/Exposure는 `fxAcct.rows()` 분기(L1643-1644, L1674)인데 History는 무조건 `DEMO_HIST`(L1708) — 실계좌 사용자가 자기 것 아닌 청산 기록 2건을 봄. **정직성 결함, 공개 전 수정 1순위.**
5. **Exposure가 비FX에 FX 계약크기 적용**: L1677 `FX_CONTRACT`(100k) 고정 — 주식/크립토 포지션이 있으면 통화 노출이 10만 배 왜곡. Trade 탭은 `FX_CONTRACT_OF`(L2034)로 올바름. 또한 base/quote 3글자 슬라이스(L1676)라 AAPL 같은 심볼이 "AAP/L"로 쪼개짐.
6. **DEMO 계좌 Balance 하드코딩 100000** (L1478) vs Mailbox·Navigator의 계좌번호 3172308·"99992.79"(L1411 FX_EQUITY는 미사용 상수) — 데모 서사 숫자가 파일 안에서 3갈래.
7. **툴바 'New Order (F9)' 아이콘이 죽어 있음**: TOOLS엔 있으나(L1840) TOOL_ACT(L1871-1885)에 없어 클릭하면 "coming in build" 토스트 — F9 키는 되는데 같은 기능 아이콘은 안 됨. `openOrderDialog` 한 줄 연결감.
8. **Trade 탭 심볼 소문자 + 접미사** (`eurusd.fx` L1384-1385, L1508). MT5는 대문자 심볼(브로커 접미사는 있어도 대문자). [브로커편차]지만 소문자는 MT5스럽지 않음.
9. **MW Spread 표기**: FX를 소수 1자리 pip("1.0", L1213)로 표시. MT5는 **정수 포인트**(예: 10). Time 컬럼도 로컬 시각(L1277) — MT5는 서버(호가) 시각.
10. **차트 그리드 = 가로선만** (L973-975). MT5 그리드는 세로+가로 점선.
11. **더블클릭 판정 450ms 수동 구현**(L1947-1953) — 이유가 주석에 명시돼 있고 동작 자체는 MT5와 동일(더블클릭=주문창) ✓. 단 MT5는 옵션에 따라 더블클릭=차트인 설정도 있음[브로커편차].
12. **차트 컨텍스트 메뉴 축소판**(L818-840): 구성 항목은 MT5 부분집합으로 정직하나, MT5의 Trading/DoM/One-Click/Templates/Colors/Save Picture 부재(1절 참조). "차트타입·TF는 우클릭에 없음" 판단(L802 주석)은 실제 MT5와 일치 ✓.
13. **볼륨 홀드 가속 ×10**(L1093-1096, L1499-1507): MT5는 홀드 연타는 있으나 가속 배율은 없음[ESTIMATE]. 무해한 개선.
14. **알림(Alerts) 1회 발화 후 무장해제**(L1406-1408): MT5 알림은 반복 횟수·간격 설정 + 사운드/메일 액션 선택. 우리는 조건 2종(≥/≤)·1회·무음.
15. **차트 창이 자유 플로팅 위젯**: MT5 기본은 최대화된 MDI + 하단 차트 탭 스트립. 우리는 항상 창모드 타일(L1164-1177) — Tile/Cascade/최소·최대화는 있어 기능적 등가지만 "차트 탭 바"가 없어 스샷 비교 시 다름.
16. **원클릭 바 위치·구성**: MT5는 좌상단 컴팩트 SELL|lot|BUY 세로 2줄 패널 + 접기 화살표. 우리는 가로 바 + 스프레드 + OHLCV(L931-937). 기능 동일, 생김새 다름. OHLCV를 크로스헤어 호버 시에만 표시(L1065-1066)는 사장님 지시 반영이나 MT5는 항상 창 좌상단에 심볼·OHLC 표기.
17. **테마 전환·위젯 추가(＋)·리셋 버튼**(L469-476): MT5에 없는 헤더 컨트롤 — 레전드 셸 고유. 100% 카피 관점에선 이질 요소이나 제품 결정 사항.

## 3) '라이브 공개 가능?' 판정 — MT5 사용자 경험 기준

**결론: 데모 계정 "조회용"(read-only 쇼케이스)으로는 조건부 공개 가능. "MT5 대체 터미널"로 소개하면 3분 내 이탈 요인이 명확히 존재.**

### 지금도 통하는 것 (공개 자산)
차트 4종 타일+실봉(Binance/fx-prices, L896-928)·라이브 시세(WS+Realtime, L1985-2029)·MW 4서브탭·컬럼 토글·드래그→차트 심볼 교체·더블클릭 주문창·원클릭 매매(DEMO)·Trade 탭 실시간 마진/스왑/플로팅·Exposure 파이·Toolbox 도크 리사이즈/접기 — **첫인상 1분은 MT5로 보인다.**

### 3분 내 이탈 트리거 (공개 전 최소 수선 목록, 우선순위순)
1. **차트에 시간축이 없다** — 트레이더가 "언제 봉인지"를 못 봄. 스샷 비교로도 즉시 들통 (1-B).
2. **LIVE 로그인인데 History가 남의 데모 기록** (2-4) — 실계좌 조회용 공개의 신뢰를 직접 깨는 유일한 "거짓 데이터". 최소한 LIVE 시 빈 상태 표시로.
3. **Ctrl+T/Ctrl+N이 브라우저 새탭/새창을 연다** (1-E) — MT5 근육기억이 바로 부딪힘. F키 대체 매핑 or 안내 필요.
4. **트레이드 레벨 드래그·중클릭 룰러·←/→ 스크롤 부재** — MT5 유저의 무의식 조작 3종이 전부 무반응.
5. **소리 0** — 주문 체결·알림이 무음이면 "데모라 대충 만들었네"로 읽힘.
6. **죽은 컨트롤 다수가 토스트**: 툴바 절반(One-click·New Order 아이콘·Data window·Screenshot·Shapes 등)·MW 우클릭 대부분·F8/Ctrl+D/Ctrl+B — "coming in build" 정직 표기는 개발용으론 옳으나 **공개용으론 죽은 버튼을 숨기는 편이 MT5답다**.
7. **Sign in 아바타 무반응** (1-F) — 계좌 조회용 공개의 입구가 없음.

### 공개 판단 가이드
- **지금 상태 그대로**: 내부 시연·사장님 리뷰·"프리뷰" 라벨 공개 = 가능.
- **고객 노출(데모 조회)**: 위 1·2·7(시간축 / LIVE History / 로그인 입구) 3건은 수정 후. 3~6은 1주 내 후속.
- **"MT5와 동일" 마케팅**: 1절 [필수] 전체(Properties·DoM·Data Window·메뉴바·레벨 드래그·룰러·사운드·원클릭 스위치) 완료 전 금지.
