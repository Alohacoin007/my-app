## ⚖️ 법적 경계 (필수 준수)
- **차용 금지:** "Robinhood", "Legend", 깃털 로고, 마스코트, 마케팅 문구, 스크린샷 자체의 복제. 상표·트레이드드레스 침해 영역.

## 2. 헤더 해부도

확정 사실 (출처: https://robinhood.com/us/en/support/articles/layouts-on-legend/ , https://robinhood.com/us/en/support/articles/widgets-in-robinhood-legend/ , https://x.com/AskRobinhood/status/1999145111041532391 , https://robinhood.com/us/en/support/articles/auras-on-legend/ ):
- **좌측: 레이아웃 탭 열 + `+` 버튼.** 탭은 최대 **9개**, `+`는 화면 좌상단, 탭마다 **아이콘 변경 가능**, **드래그로 순서 변경**, 우클릭 메뉴에 **Duplicate layout / Open in new tab**.
- **상단 중앙(레이아웃 화면 상단): Aura 타이틀** — 클릭하면 아우라(배경 테마) 변경.
- **우측: `Add widget` 버튼** — "top-right corner of your layout"에 위치(공식 문구). 클릭 → 위젯을 자동 배치하거나 원하는 자리에 드래그.

구조 스펙 (수치는 전부 [ESTIMATE], 근거: 48px 헤더 관례 + 스크린샷 비례):

```
┌────────────────────────────────────────────────────────────────────────┐
│ [로고 20px] [탭1][탭2]…[+]      [Aura/상태 필]      [Add widget][🔔][⚙][아바타] │ h=48px
└────────────────────────────────────────────────────────────────────────┘
```

| 요소 | 값 | 상태 |
|---|---|---|
| 헤더 높이 | 48px [ESTIMATE] | — |
| 레이아웃 탭 | h=32px, padding 0 12px, radius 8px(상단만) 또는 필형 [ESTIMATE]. 아이콘 14px + 제목 13px | 활성: surface색 + ink 텍스트 / 비활성: 투명 + dim 텍스트 / 호버: rgba(255,255,255,0.06) [ESTIMATE] |
| `+` (새 레이아웃) | 32×32px, 라인 + 아이콘, dim→ink 호버 [ESTIMATE] | 최대 9탭 도달 시 비활성 (탭 수는 확정) |
| 중앙 필 | Aura 타이틀(확정 요소). 시장 개장 상태 표시가 이 자리인지는 미확정 → §7 | [ESTIMATE] h=28px 필, pill 배경 |
| `Add widget` | 확정 위치=우상단. [ESTIMATE] h=32px, 라벨+아이콘, 강조 스타일(액센트 또는 밝은 보더) | 호버: 밝기 +6% |
| 알림·설정·아바타 | [ESTIMATE] 20px 아이콘 3개, 간격 16px, 아바타 28px 원형 | 공개 문서에 미기재 → §7 |

**ALPEXA 매핑:** 좌=대시보드 탭(스포츠/카지노/계좌)+`+`, 중=경기일·오픈 상태 필, 우=위젯 추가·알림·설정·아바타.

## 3. 위젯 크롬 해부도

확정 사실 (출처: https://robinhood.com/us/en/support/articles/widget-linking-on-legend/ , https://robinhood.com/us/en/support/articles/widgets-in-robinhood-legend/ ):
- 타이틀바 **좌측 끝 = 사각형(square) 링크 아이콘**. 클릭 → 색상 선택 → **다른 위젯에서 같은 색을 고르면 링크(동기 그룹)**. `Unlinked` 선택으로 해제.
- 타이틀바 **우측 = 세로 점 3개(⋮) 메뉴**: `Group by`(자산/자산유형별 그룹), `Customize columns`(옵션체인 데이터포인트 추가/제거) 등.
- 위젯은 **추가·리사이즈·리포지션·커스터마이즈** 전부 가능(공식 문구), 커스터마이즈는 **자동 저장**.
- 차트 위젯 한정: 우상단에 **지표 아이콘**(indicator icon)과 **차트 타입 아이콘**.
- 트레이딩 위젯 한정: **번개 아이콘 = Auto-send 토글**, **원+빗금 = Exit & Cancel all**(해당 심볼 전체 청산+주문취소).
- **Draw syncing**: 한 차트의 드로잉이 같은 심볼의 모든 차트에 자동 표시.

구조 스펙:

```
┌─[■링크색]─[제목/심볼]─[검색]───────────[⚡][지표][차트타입][⋮]─┐  타이틀바 36px [ESTIMATE]
│                                                              │
│                     위젯 본문 (padding 12px [ESTIMATE])          │
│                                                              │
└──────────────────────────────────────────────────────⤡──────┘  우하단 리사이즈 핸들
```

| 항목 | 값 |
|---|---|
| 링크 사각형 | 12×12px, radius 3px, 선택된 링크색으로 채움; 미링크 시 아웃라인만 [ESTIMATE]. 위치=타이틀바 좌측 끝(확정) |
| 링크 색상 수 | **미확정** — 메커니즘(색으로 그룹핑)만 공식. 4색 가정 [ESTIMATE] → §7 요청 |
| 보더/구분 | 보더 없음, `bg`와 `surface` 명도차 + widgetGap 8px로 구분 [ESTIMATE] |
| 라운드 | 8px [ESTIMATE] |
| 그림자 | 정지 상태 없음, 드래그 중에만 [ESTIMATE] |
| 드래그 | 타이틀바 잡고 이동, 자동 배치 또는 자유 드롭(확정: "click to add … or drag the widget to your desired spot") |
| 리사이즈 | 모서리/변 핸들 [ESTIMATE], 최소 크기 위젯별 상이 [ESTIMATE] |
| 활성 위젯 | 키보드 단축키는 "마지막 선택한 위젯(active widget)"에 적용(확정) → 활성 위젯에 시각 표시(보더 하이라이트) 필요 [ESTIMATE] |

### 4.1 차트
- **한 창에 차트 최대 8개** (확정: https://newsroom.aboutrobinhood.com/the-legend-awakens/ )
- **차트 종류** (확정 일부): Candle, Line, Heikin-Ashi, Hollow candles + "several chart types … and more"(공식). Bar/Area/Baseline은 일반 차트 문서에서 언급 [준확정] (출처: widgets-in-robinhood-legend, using-advanced-charts)
- **인터벌**: 서브초 실시간 갱신 + 일·시·분·초 커스텀 인터벌 (확정: the-legend-awakens)
- **지표**: 런칭 시 "dozens"/50+ → **현재 90+** (확정: hood-summit-2025-news). 확정 명시된 지표: RSI, MACD, Bollinger Bands, 이동평균(EMA 등), VWAP, **Auto-anchored VWAP**, Ichimoku Cloud, Keltner Channels, Parabolic SAR, Fast Stochastic, On-Balance Volume, Awesome Oscillator, Aroon Oscillator, ATR, Williams %R(+"9 other momentum indicators", 공식 X: https://x.com/RobinhoodApp/status/1885044646989922723 ). 전체 목록은 https://robinhood.com/us/en/support/articles/chart-indicators-on-legend/ (프록시 차단으로 전문 미수집 → §7)
- **지표 세트 저장**: 지표 조합을 저장해 여러 차트에 일괄 적용 (확정)
- **드로잉 도구** (확정 일부): Trend line, Horizontal line, Fibonacci retracement, Channels, **Anchored VWAP**(앵커 지점 직접 선택). Rectangle 등 나머지 미확정 → §7
- **드로잉 동기화**: 같은 심볼의 모든 차트에 드로잉 자동 반영 (확정)
- **Level 2**: 차트 위에 실시간 매수/매도 관심 시각화, **$5/월** (출처: https://robinhood.com/us/en/newsroom/nasdaq-level-2-market-data-is-here/ )

### 4.2 주문
- **주문 타입** (확정): Market, Limit, Stop, Stop-limit, Trailing stop + 지원 종목 시간외(Extended hours) 거래 (출처: finder.com/investments/robinhood-legend , robinhood.com/us/en/support/articles/legend-behaviors-and-settings/ )
- **기본값** (확정: legend-behaviors-and-settings):
  - Time-in-force 기본 = **GFD(good-for-day)**
  - Stop 가격 기본 = 매도 시 bid −1% / 매수 시 ask +1%
  - Trailing stop 기본 = percentage 방식, **1%**
  - 기본 수량 = 주식 1주 / 크립토 1코인 / 선물 1계약
- **Auto-send** (확정): 번개 토글 ON → 확인 화면 생략 즉시 전송(표준 주문 체크 후), OFF → 프리필된 주문폼 오픈. ON 상태에선 수수료 사전 표시 불가.
- **Exit & Cancel all** (확정): 원+빗금 버튼, 선택 심볼의 전 포지션 청산 + 전 대기주문 취소.

3. **탭 7종 복원**(초안 NCAAB 누락) + 터치 높이 40px 보정(T5).
